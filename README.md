# API iTaaS Solution

This is a API to convert CDN Logs to Agora Format

## Start

Install :
- NodeJs
- NPM or Yarn

```bash
npm install
```

Start Project
```bash
npm run start
```

## Modules
- TypeScript
- Express
- Axios
- Joi
